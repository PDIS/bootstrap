色彩對比度

在替所製作的數位系統選用色彩時，我們必須符合色彩對比度 WCAG AA 等級或更高級別。以下列出幾個可以用來檢查對比度是否足夠的工具： Snook.ca 顏色對比檢查，Webaim 顏色對比檢查器、TPG 顏色對比分析器、對比度網站-Contrast ratio 或 Android 無障礙掃描工具。

要注意的是，字型大小也會是影響對比度的變因。舉例來說，有時雖然文字色與背景色通過上述色彩對比度檢測，但若文字的字型縮小時，雖顏色不變，可能必須得將字重加大才能通過對比度檢測。您可以使用下列工具來檢查：Adobe 色彩對比度分析器。
